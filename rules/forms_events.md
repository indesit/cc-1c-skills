# Добавление обработчиков событий на форму 1С

**ВАЖНО!** — при добавлении обработчика события в модуль формы, не забудь также добавить привязку события в XML-файл формы.
Файл обычно называется `Form.xml` и находится в родительском каталоге относительно модуля формы.

Привязки событий в XML выглядят так:

```xml
<Events>
	<Event name="OnOpen">ПриОткрытии</Event>
	<Event name="BeforeWrite">ПередЗаписью</Event>
	<Event name="OnCreateAtServer">ПриСозданииНаСервере</Event>
</Events>
```

## Типовые события формы:

| XML Event Name | Имя обработчика | Описание |
|----------------|-----------------|----------|
| `OnOpen` | ПриОткрытии | Клиент, при открытии формы |
| `OnClose` | ПриЗакрытии | Клиент, при закрытии формы |
| `BeforeWrite` | ПередЗаписью | Клиент, перед записью |
| `AfterWrite` | ПослеЗаписи | Клиент, после записи |
| `OnCreateAtServer` | ПриСозданииНаСервере | Сервер, создание формы |
| `BeforeWriteAtServer` | ПередЗаписьюНаСервере | Сервер, перед записью |
| `AfterWriteAtServer` | ПослеЗаписиНаСервере | Сервер, после записи |
| `OnReadAtServer` | ПриЧтенииНаСервере | Сервер, при чтении объекта |

Значение внутри тега `<Event>` — это имя процедуры-обработчика в модуле формы.
