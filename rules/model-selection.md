# Стратегия выбора моделей

Распределение моделей по ролям.

Применяется: при работе с AI-ассистентами (Claude Code, Cursor, другие IDE).

---

## Принцип

**Opus — основная рабочая модель. Sonnet — только для кросс-ревью.**

Вся работа выполняется на Opus: анализ, кодогенерация, исследование, рефакторинг. Sonnet используется исключительно как вторая модель для кросс-ревью (план, код), обеспечивая независимый взгляд.

---

## Распределение

### Opus — все задачи

| Категория | Примеры |
|-----------|---------|
| **Архитектура** | Проектирование, выбор технологий, breaking changes, миграция |
| **Безопасность** | Ревью кода, аудит RLS, поиск уязвимостей |
| **Production-баги** | Порча данных, race conditions, регрессии |
| **Кодогенерация** | Формы, тесты, запросы, скрипты, конфигурации |
| **Рефакторинг** | Упрощение, извлечение методов, консолидация |
| **Исследование** | Навигация по коду, поиск, документация |
| **Итеративная работа** | Баг-фиксы, реализация фич, тестирование |

### Sonnet — кросс-ревью

| Когда | Что делает |
|-------|-----------|
| После завершения кода | Ревью изменённых файлов (находки, severity, вердикт) |
| После составления плана | Ревью плана (полнота, порядок, риски) |

Sonnet запускается через Agent/Task с `model: "sonnet"` — это обеспечивает независимую проверку другой моделью.

---

## Принципы экономии контекста

### Инкрементальная загрузка

Вместо чтения целого файла — читать нужный фрагмент (сигнатуру, метод). Расширять при необходимости.

### Изоляция через агентов

Параллельные задачи запускать через агентов — каждый со своим контекстом. В основной чат возвращается только результат.

### Батчинг операций

Вместо 10 отдельных вопросов — 1 батч. Вместо последовательных агентов — параллельные.

### Переиспользование результатов

Результаты исследования, принятые решения, найденные паттерны — сохранять и переиспользовать, не перечитывать заново.

## Антипаттерны

| Антипаттерн | Правильно |
|-------------|-----------|
| Sonnet для основной работы | Opus |
| Opus для кросс-ревью своего же кода | Sonnet (независимый взгляд) |
| Чтение всего модуля целиком при поиске одного метода | Читать конкретный метод |
| Последовательный запуск 5 агентов | Параллельный запуск |
| Продолжение чата при огромном контексте | Новый чат с сохранением ключевых решений |
