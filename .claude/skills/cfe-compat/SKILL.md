# /cfe-compat — Совместимость расширения с базовой конфигурацией

Статический анализ XML-исходников расширения против XML-дампа базовой конфигурации. Отвечает на вопрос: **«сломается ли расширение на этой версии конфигурации?»** — главный риск при обновлении вендорской конфигурации.

## Что проверяется

| Проверка | Уровень |
|----------|---------|
| Заимствованный объект (`ObjectBelonging=Adopted`) существует в базе | ERROR |
| Заимствованная форма существует в базе | ERROR |
| Метод из `&Перед/&После/&Вместо("X")` существует в базовом модуле | ERROR |
| Число параметров перехватчика = числу параметров базового метода | WARNING |
| Вызовы функций, не найденных ни в расширении, ни в базовом модуле | WARNING |

WARNINGS «unresolved-call» могут содержать платформенные методы, не попавшие в allowlist — проверяй их по смыслу, это кандидаты на «функцию убрали из базового модуля».

## Типовой сценарий: проверка перед обновлением

1. Получи дамп НОВОЙ конфигурации (`/db-dump-xml` с тестовой базы с новым релизом, или из поставки).
2. Прогони `/cfe-compat` каждого расширения против нового дампа.
3. ERRORS — расширение сломается, чинить до обновления. WARNINGS — проверить вручную.

## Команда

Движок — Python (`scripts/cfe-compat.py`); ps1 — обёртка.

```powershell
python .claude/skills/cfe-compat/scripts/cfe-compat.py -ExtensionPath <dir> -ConfigPath <dir> [-Json report.json] [-Strict]
```

### Параметры

| Параметр | Обязательный | Описание |
|----------|:------------:|----------|
| `-ExtensionPath <путь>` | да | Каталог XML-исходников расширения (с Configuration.xml) |
| `-ConfigPath <путь>` | да | Каталог XML-дампа базовой конфигурации |
| `-Json <файл>` | нет | Сохранить отчёт в JSON |
| `-Strict` | нет | WARNINGS тоже дают код возврата 1 |

## Коды возврата

| Код | Описание |
|-----|----------|
| 0 | COMPATIBLE (возможно с warnings) |
| 1 | INCOMPATIBLE (errors) или warnings при `-Strict` |

## Примеры

```powershell
python .claude/skills/cfe-compat/scripts/cfe-compat.py -ExtensionPath "src\kassir-avtokassa" -ConfigPath "src\baf-config-dump"
python .claude/skills/cfe-compat/scripts/cfe-compat.py -ExtensionPath "src\my-ext" -ConfigPath "C:\dumps\new-release" -Json compat.json -Strict
```
